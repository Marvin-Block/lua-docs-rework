# c_config

## Functions

### add_int

| Argument | Type                                    | Required |
| -------- | --------------------------------------- | -------- |
| value    | `number`                                | +        |
| var_name | `string`                                | +        |
| returns  | [`config_var_t`](/structs/config_var_t) |          |

```lua
local some_name = g_config:add_int(100, "some_name")

-- set config var value to 200
some_name:set_int(200)

print("Value is: " .. some_name:get_int()) -- prints 200
```
