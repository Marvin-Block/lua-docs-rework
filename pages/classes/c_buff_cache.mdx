# c_buff_cache

## Functions

### get_buff

| Argument     | Type                        | Required |
| ------------ | --------------------------- | -------- |
| object_index | `number`                    | +        |
| name         | `string`                    | +        |
| returns      | [`c_buff`](/classes/c_buff) |          |

```lua /get_buff/
local buff = features.buff_cache:get_buff(g_local.index, "asheqcastready")

if buff and buff:get_stacks() == 4 then
    print("Ashe has 4 stacks of her Q")
end
```

### has_buff_of_type

Check if object as a buff of a specific type.

| Argument     | Type                                | Required |
| ------------ | ----------------------------------- | -------- |
| object_index | `number`                            | +        |
| type         | [`e_buff_type`](/enums/e_buff_type) | +        |
| returns      | `boolean`                           |          |

```lua /has_buff_of_type/
local is_blind = features.buff_cache:has_buff_of_type(g_local.index, e_buff_type.blind)

if is_blind then
    print("I'm blind!")
end
```

### get_all_buffs

This function returns all cached buffs for an object.

| Argument     | Type                               | Required |
| ------------ | ---------------------------------- | -------- |
| object_index | `number`                           | +        |
| returns      | [`table<c_buff>`](/classes/c_buff) |          |

```lua /get_all_buffs/
local buffs = features.buff_cache:get_all_buffs(g_local.index)

-- Print all buffs on the local player
for _, buff in pairs(buffs) do
    print(buff:get_name())
end
```
